About

I have created this bot to solve some manual work for moderator as well as admins. Also i have added some features in this bot. There are lot more features i am developing currently which will be released in future times..

!ping

!ping commands responds with pong

!beep

!beep commands responds with boop

!args-info

!args-info commands responds with arguments passed to command

!avatar

!avatar || !avatar @user commands responds with avatar of a tagged user, if no user is tagged it will dislay avatar of a user who ran the command

!server

!server commands responds with server name

!user-info @taggedUser

!user-info @user commands responds with user information of a tagged user

!prune

!prune (any number) commands deletes the number of comments from the channel

!help

!help commands DM you with the list of commands it has
